As part of my Pattern Recognition course, I worked on the Palmer Penguins dataset to explore biological differences between species using data visualization and statistical analysis.

🔍 What I did:

Performed full EDA (histograms, pairplots, boxplots, heatmaps)
Applied One-Way ANOVA to compare flipper length across species
Used Chi-Square test to examine the relationship between species and body mass

📊 Key insights:

Clear differences between penguin species in body mass and flipper length
Statistical tests confirmed that these differences are significant

This project helped me better understand how to move from raw data to meaningful insights using Python and statistics.
